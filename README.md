# CoursesApp

Тестовое Android-приложение по ТЗ «Тысячи курсов». Показывает каталог онлайн-курсов, позволяет авторизоваться, просматривать список, сортировать и отмечать избранные.

## Функциональность

- **Экран входа**
    - Ввод e-mail и пароля.
    - Валидация e-mail по маске `текст@текст.текст` без кириллицы.
    - Кнопка «Вход» активна только при валидном e-mail и непустом пароле.
    - «Регистрация» и «Забыл пароль» — заглушки.
    - Кнопки VK/OK открывают соответствующие сайты в браузере.
    - При успешном нажатии «Вход» — переход на главный экран.

- **Главный экран**
    - Список курсов из мок-API (JSON в Google Drive).
    - Краткая карточка курса: обложка (заглушка), название, описание (до 2 строк), цена, рейтинг, дата публикации.
    - Формат даты: `2024-05-22` → `22 мая 2024`.
    - Сортировка по дате публикации.
    - Добавление/удаление курса из избранного (иконка закладки).

- **Экран «Избранное»**
    - Отображает только курсы, помеченные как избранные.
    - Плейсхолдер-сообщение, если избранных курсов нет.

- **Нижняя навигация**
    - Три вкладки: «Главная», «Избранное», «Аккаунт».
    - Экран «Аккаунт» реализован как заглушка.
    - Всегда видна на экранах после авторизации.

## Технологии и архитектура

- **Язык:** Kotlin
- **Минимальная версия Android:** API 26 (Android 8.0)
- **Модули (многомодульная фича-архитектура):**
    - `app` — точка входа, подключает фичи/core/data/domain.
    - `core-ui` — общие ресурсы (стили, цвета, иконки, шрифты) и утилиты (`ResourceProvider`).
    - `feature-auth` — экран логина.
    - `feature-main` — `MainActivity`, bottom navigation, nav-graph.
    - `feature-home` — список курсов, адаптеры, `MainViewModel`/`MainUiState`.
    - `feature-favorites` — экран избранного.
    - `feature-account` — заглушка профиля.
    - `domain` — бизнес-модели, интерфейсы репозиториев, use case’ы.
    - `data` — работа с сетью и базой данных (Retrofit, Room).
- **UI:**
    - XML-разметка, ViewBinding.
    - Material Components, кастомная цветовая схема и типографика.
    - BottomNavigationView + Navigation Component.
- **DI:** Hilt (+ KSP)
- **Навигация:** Navigation Component (NavHostFragment + граф навигации)
- **Сеть:** Retrofit, OkHttp, Kotlinx Serialization
- **Данные:**
    - Room — хранение курсов и состояния избранного.
- **Асинхронность:** Kotlin Coroutines, StateFlow (MVVM-подход)

## Запуск

1. Открыть проект в **Android Studio** (Arctic Fox / Giraffe и выше).
2. Дождаться синхронизации Gradle.
3. Собрать и запустить конфигурацию `app` на устройстве или эмуляторе с Android 8.0+ (API 26+).
